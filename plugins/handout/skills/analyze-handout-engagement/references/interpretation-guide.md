# Engagement interpretation guide

| Observation | Reasonable interpretation | Avoid claiming | Proportionate follow-up |
|---|---|---|---|
| Recent site visit | The shared link was opened | A named person read it | Offer help and reference the shared material |
| Repeated page navigation | That topic may deserve clarification | The buyer has a confirmed priority | Ask whether the topic is relevant or needs detail |
| CTA or external-link click | A specific destination was opened | The action was completed | Follow up on the destination or next step |
| Longer session | More activity occurred in one session | Attention quality or positive sentiment | Send a concise, content-specific check-in |
| Multiple recipient variants active | Several shared links generated activity | The buying committee is aligned | Map stakeholders and ask about the process |
| No recorded activity | No matching tracked activity was found | The buyer ignored the site | Verify delivery, filters, consent, and timing |

Prefer high confidence for exact recorded events, medium confidence for behavioral patterns, and low confidence for intent. Make those levels visible when a recommendation depends on inference.
